# 🌱 EcoPrompt AI

EcoPrompt AI is an AI-powered prompt optimization platform that helps reduce unnecessary AI computation by creating shorter, clearer, and more efficient prompts.

## Features

- AI-powered Prompt Optimization
- Efficient AI Response Generation
- Eco Score Calculation
- Energy Saving Estimation
- CO₂ Reduction Estimation
- Sustainability Dashboard

## Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Google Gemini AI
